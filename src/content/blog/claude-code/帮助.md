# 第一部分：原生 Claude Code 命令

以下是 Claude Code 官方原生支持的命令。

---

## 一、核心命令速查（最常用）
### 1. `/compact` — 压缩上下文（必学）
**作用**：精简对话历史，保留关键信息，防止模型失忆、节省 Token
**用法**：
```
/compact                          # 自动压缩
/compact [保留什么内容]           # 带指令精准压缩
```

**常用指令模板（直接复制）**
```
/compact 保留最终代码、文件修改、核心逻辑
/compact 保留架构决策、API 定义、数据库变更
/compact 保留报错原因、解决方案、关键步骤
/compact 保留需求要点、待办事项、最终结论
/compact 只保留重要信息，删除试错、闲聊、草稿
```

---

### 2. `/clear` — 清空上下文
**作用**：彻底清空对话历史，重新开始
**用法**：
```
/clear
```

---

### 3. `/export` — 导出当前对话
**作用**：把完整对话导出成文件，备份、存档、分享
**用法**：
```
/export           # 导出完整对话
```

---

### 4. `/reset` — 重置对话
**作用**：重置整个对话，清除所有上下文和任务状态
**用法**：
```
/reset
```

---

## 二、文件/项目命令（开发必备）
### 5. `/files` — 查看当前项目文件
**作用**：列出工作区所有文件
**用法**：
```
/files
```

---

### 6. `/open [文件]` — 打开文件
**作用**：直接打开指定文件
**用法**：
```
/open main.py
/open src/app.js
/open README.md
```

---

### 7. `/create [文件]` — 创建文件
**作用**：新建一个文件
**用法**：
```
/create utils.js
/create config.json
```

---

### 8. `/delete [文件]` — 删除文件
**作用**：删除工作区文件
**用法**：
```
/delete temp.txt
```

---

### 9. `/run` — 运行代码
**作用**：执行当前项目代码
**用法**：
```
/run
/run index.js
/run main.py
```

---

### 10. `/git` — Git 快捷操作
**作用**：快速执行 Git 命令
**用法**：
```
/git status
/git add .
/git commit -m "更新"
/git push
```

---

## 三、撤销操作
### 11. 基础撤销命令
```
/undo        # 撤销上一步编辑
/redo        # 恢复上一步
```

> 其他类似 `/format`、`/lint`、`/debug`、`/todo` 等命令通常由扩展或自定义技能提供，不是原生命令。

---

## 四、配置与信息
### 12. 信息与配置命令
| 命令 | 作用 |
|------|------|
| `/help` | 显示帮助信息 |
| `/model` | 列出并切换可用 AI 模型 |
| `/cost` | 查看当前会话 Token 使用量和估算费用 |
| `/config` | 打开配置设置 |
| `/version` | 显示 Claude Code 版本 |
| `/exit` `/quit` | 退出 Claude Code |

---

## 五、最常用组合（直接背这 5 条）
```
/compact 保留关键信息，删除试错过程
/open 文件名
/run
/export
/clear
```
